
## Full Stack Developer Intern Assignment(GIRMAN TECHNOLOGIES)(Next.js):

* Note: My first Next.js App *

To run in your local machine:

1. Clone the project: git clone URL
2. Move to project directory:
3. Run the dev server: npm run dev


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

*1. Navbar page Content: This component can be used as a web application's main navigation and search bar, offering a clean interface for searching and viewing user details. It adapts to different screen sizes, providing a user-friendly experience on mobile and desktop devices.*

*2. Page(Search) page Content: This part functions as a search page where users can enter a query to locate and display user data. It offers an interactive and responsive user interface with modals for comprehensive user data and a fallback image if there is no data.*
